# PWA2Native 🚀

Convert any Progressive Web App (PWA) into installable `.apk` (Android) and iOS packages with a simple interface.

## 🌟 Features

- **PWA to APK:** Generate Android packages (TWA or WebView) from a URL.
- **PWA to iOS:** Generate iOS project files ready for Xcode.
- **Simple UI:** A clean dashboard to configure your app's manifest, icons, and theme colors.
- **Open Source:** Built with modern tools and community-driven.

## 🛠️ Built With

- **Framework:** [Next.js](https://nextjs.org/)
- **Core Engine (Android):** [Bubblewrap](https://github.com/GoogleChromeLabs/bubblewrap)
- **Core Engine (iOS/Android):** [Capacitor](https://capacitorjs.com/)
- **Styling:** CSS Modules / Tailwind

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/guistela/PWA2Native.git
   cd PWA2Native
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Follow the UI to input your PWA URL and generate your app!

## 📦 Requirements

- **Android Studio** (for building APKs)
- **Xcode** (for building iOS apps - macOS only)
- **Node.js** (v18+)
